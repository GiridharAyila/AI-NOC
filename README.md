# AI-Assisted Enterprise Network Operations Center (AI-NOC)

An enterprise-grade Cisco networking project combining CCNA concepts, network monitoring, automation, and AI-assisted troubleshooting.

## Project Goals

- Design an enterprise network using Cisco Packet Tracer
- Implement VLANs, Routing, DHCP, DNS, ACLs, and SSH
- Develop a network monitoring system using Python
- Create an AI-assisted troubleshooting engine
- Build a dashboard for network health monitoring
